# Project Features

This project is a complete online internship portal for AH Nexues Lab.

## Core modules

1. Student registration
2. Company registration
3. Login/logout
4. Internship listing
5. Internship posting
6. Internship application
7. Application status tracking
8. Saved internships
9. Student profile
10. Notifications
11. Interview schedule
12. Certificate page
13. Resume builder
14. Skill quiz
15. Compare internships
16. Admin analytics

## Backend database tables

- users
- internships
- applications
- notifications
- activities
- quiz_scores

## Concepts used

### OOP
Frontend JavaScript uses classes:
- APIService
- Validator
- ApplicationQueue
- ActivityStack
- SearchSorter

Backend PHP uses reusable functions and API modules:
- auth.php
- data.php
- config.php

### PF
- Variables
- Arrays
- Functions
- Loops
- Conditions
- Form handling
- Validations

### DSA
- Queue for company applications
- Stack for recent activities
- Searching internships
- Insertion sorting internships

### Database
- MySQL tables
- Primary keys
- Foreign keys
- CRUD
- Login sessions
- Password hashing
